# Installing

Copy the **.jar** file to your plugins folder. Run the server.
Now, go to the **config.conf**.  
You will see the following:  

```kotlin
database {
  type: "MySQL"
  host: "localhost"
  database: "eastereggs"
  port: 3306
  user: "eastereggs"
  password: "123123"
}

languageFile = "en"
```

To get help about the plugin type `/ee help`.
